# How to Install the Ansible:


- Follow these steps for the Ubuntu 22.04"
    ```
    sudo apt update
    sudo apt install software-properties-common
    sudo add-apt-repository --yes --update ppa:ansible/ansible
    sudo apt install ansible
    ```

- Follow these steps for Windows:
    You can't directly run the Ansible on the windows, but can be run under the WSL 
    > [How to install the WSL on Windows](https://learn.microsoft.com/en-us/windows/wsl/install-manual)
    > [How to configure the WSL for Ansible](https://phoenixnap.com/kb/install-ansible-on-windows)
